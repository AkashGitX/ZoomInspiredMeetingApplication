🚀 Zoom-Inspired Meeting App

✨ A full-stack video conferencing web app with secure authentication and real-time communication.

🔥 Features :

🔐 Secure Login & Signup (Spring Security + JWT)
👥 User Management (Online/Offline status)
🎥 Create & Join Video Meetings
🔗 Share Meeting Links
💬 Chat, Screen Share & Media Controls


⚙️ Tech Stack :

☕ Spring Boot + Spring Security + JWT
🐘 PostgreSQL + Spring Data JPA
🌐 HTML, CSS, JavaScript
📡 ZegoCloud WebRTC


🔄 Flow :

Register → Login → Dashboard → Video Meeting
💡 Highlights
🔐 Secure authentication
🎥 Real-time video integration
🗄️ Database-driven system
🎯 Clean & responsive UI
